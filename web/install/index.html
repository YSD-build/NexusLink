NexusLink

一键安装部署

复制下方命令到您的服务器执行，自动完成下载、配置和启动

一键安装命令
在服务器终端执行
curl -sSL https://raw.githubusercontent.com/YSD-build/NexusLink/main/install.sh | sudo bash
温馨提示
安装完成后会自动生成随机管理员密码，请妥善保存。当前版本仅支持管理功能，内网穿透功能后续更新。
安装步骤
复制安装命令
点击上方"复制命令"按钮，复制一键安装脚本
登录服务器执行
使用 SSH 登录您的 Linux 服务器，粘贴命令并执行
等待自动安装
脚本会自动下载、配置依赖、设置开机自启，约需 1-3 分钟
访问管理面板
安装完成后，使用输出的管理地址和随机密码登录管理面板
系统要求
操作系统
Linux (systemd)
架构支持
x86_64 / ARM64
内存要求
≥ 256MB
默认端口
7001
